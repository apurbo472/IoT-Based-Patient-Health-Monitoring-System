# IoT Based Patient Health Monitoring System

## Overview

This project is an Internet of Things (IoT) based patient health monitoring system designed to continuously monitor a patient's vital health parameters and transmit the collected data for remote observation.

The system acquires sensor data in real time, processes it using a microcontroller, and displays or sends the information through an IoT platform, enabling caregivers to monitor patients remotely.

---

## Features

- Real-time health monitoring
- Continuous sensor data acquisition
- IoT-based remote monitoring
- Live display of patient parameters
- Wireless data transmission
- Modular hardware design
- Expandable architecture for additional sensors

## Software

- Arduino IDE
- Embedded C/C++
- IoT Platform (Blynk / ThingSpeak / Firebase / MQTT)

---

## System Workflow

1. Read sensor values
2. Process sensor data
3. Display health parameters
4. Send data to IoT platform
5. Monitor remotely through dashboard
